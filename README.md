# SceneForge v1 - 基础剧本编辑器

## 版本概述

SceneForge 的第一个版本，实现了剧本写作工具的核心骨架：项目结构搭建、暗色主题样式系统、本地存储模块，以及支持标题与正文编辑的基础编辑器。

## 功能清单

- **剧本标题编辑** — 顶部输入框，支持最多120字符的剧本标题
- **正文内容编辑** — 等宽字体文本区域，支持自由编写剧本内容
- **自动保存** — 输入停止1.5秒后自动将内容保存到 localStorage
- **保存状态指示** — 底部状态栏实时显示保存进度（保存中 / 已保存）
- **字符计数** — 底部实时统计正文字符数
- **数据持久化** — 页面刷新或关闭后重新打开，自动恢复上次编辑内容
- **页面关闭前保存** — 监听 beforeunload 事件，确保不丢失数据

## 技术栈

- 纯 HTML/CSS/JS，无需构建工具，直接用浏览器打开
- localStorage 实现客户端持久化
- IIFE 模块模式组织代码
- CSS 自定义属性实现暗色主题

## 文件结构

```
v1-basic-editor/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式表（暗色主题）
├── js/
│   ├── storage.js      # 存储模块（localStorage 封装）
│   └── app.js          # 主应用模块（编辑器逻辑）
└── README.md
```

## 使用方式

1. 直接用浏览器打开 `index.html`
2. 在顶部输入剧本标题
3. 在正文编辑区编写剧本内容
4. 所有修改会自动保存，刷新页面不丢失
