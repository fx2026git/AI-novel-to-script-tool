# SceneForge v1 - 基础剧本编辑器

## 版本概述

SceneForge 的第一个版本，实现了剧本写作工具的核心骨架：项目结构搭建、暗色主题样式系统、本地存储模块，以及支持标题与正文编辑的基础编辑器。

## 功能清单

- **剧本标题编辑** — 顶部输入框，支持最多120字符的剧本标题
- **正文内容编辑** — 等宽字体文本区域，支持自由编写剧本内容
- **自动保存** — 输入停止1.5秒后自动将内容保存到 localStorage
- **保存状态指示** — 底部状态栏实时显示保存进度（保存中 / 已保存）
- **字符计数** — 底部实时统计正文字符数
- **数据持久化** — 页面刷新或关闭后重新打开，自动恢复上次编辑内容
- **页面关闭前保存** — 监听 beforeunload 事件，确保不丢失数据

## 技术栈

- 纯 HTML/CSS/JS，无需构建工具，直接用浏览器打开
- localStorage 实现客户端持久化
- IIFE 模块模式组织代码
- CSS 自定义属性实现暗色主题

## 文件结构

```
v1-basic-editor/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式表（暗色主题）
├── js/
│   ├── storage.js      # 存储模块（localStorage 封装）
│   └── app.js          # 主应用模块（编辑器逻辑）
└── README.md
```

## 使用方式

1. 直接用浏览器打开 `index.html`
2. 在顶部输入剧本标题
3. 在正文编辑区编写剧本内容
4. 所有修改会自动保存，刷新页面不丢失

# SceneForge v2 - 角色管理模块

## 版本概述

在 v1 基础编辑器之上，新增**角色管理**功能模块。通过右侧功能面板，用户可以创建、编辑、删除剧本角色，每个角色支持名称、类型（主角/反派/配角/次要角色）和文字描述。页面布局从单栏改为左右分栏结构，为后续场景、时间线等模块预留标签入口。

## 新增功能

- **角色CRUD** — 添加、编辑、删除角色
- **角色属性** — 名称（必填）、角色类型（下拉选择）、描述文字
- **角色列表** — 卡片式展示所有已创建角色，显示名称、类型标签、描述
- **侧边栏面板** — 右侧功能面板，使用标签切换不同功能模块
- **标签式导航** — 角色/场景/时间线/导出四个标签（后续版本逐步启用）
- **数据持久化** — 角色数据保存到 localStorage，刷新不丢失

## 相较于 v1 的变更

| 文件  | 变更类型 | 说明  |
| --- | --- | --- |
| `index.html` | 修改  | 重构为左右分栏布局，增加侧面板和角色UI |
| `css/style.css` | 扩展  | 新增面板、标签、表单、角色卡片、按钮等样式 |
| `js/storage.js` | 扩展  | 新增 CHARACTERS 存储键 |
| `js/characters.js` | 新增  | 角色管理模块（CRUD + 渲染） |
| `js/app.js` | 修改  | 增加Characters模块初始化和标签切换逻辑 |

## 技术要点

- 角色数据使用 JSON 数组序列化存储
- 角色ID使用时间戳+随机字符串生成，保证唯一性
- 表单内联在面板中，通过显示/隐藏切换
- 事件委托处理角色列表中的编辑/删除按钮点击

## 使用方式

1. 浏览器打开 `index.html`
2. 左侧编辑区正常编写剧本
3. 右侧面板「角色」标签下，点击「+ 添加角色」
4. 填写角色名称、选择类型、输入描述后保存
5. 角色卡片出现在列表中，支持编辑和删除
